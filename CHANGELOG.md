# Changelog

All notable changes to this project are documented here.

The format follows Keep a Changelog, and this project uses semantic versioning.

## [0.1.3] - 2026-04-16

### Fixed

- Add spacing between the Python toolbar icon and `Save .py` label.

## [0.1.2] - 2026-04-16

### Fixed

- Correct project homepage and source links to the canonical GitHub repository.

## [0.1.1] - 2026-04-16

### Fixed

- Allow publishing a new patch release after the initial PyPI upload.

## [0.1.0] - 2026-04-15

### Added

- Initial JupyterLab 4 prebuilt frontend extension.
- Notebook toolbar button and command palette command named `Save .py`.
- Frontend notebook-to-Python conversion with `percent` and `script` formats.
- Contents API save into the active notebook's server-side directory.
- Configurable overwrite behavior.
- PyPI-ready Python packaging.
